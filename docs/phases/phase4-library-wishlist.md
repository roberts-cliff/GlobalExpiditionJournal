# Phase 4: Library & Wishlist

## Backend (Go)

- Library API (saved articles, facts, links)
- Tagging system for cross-referencing
- Wishlist API (planned destinations)
- Pre-trip checklist generation
- Search/filter across library items

## Frontend (Angular)

- Library page with search and categories
- Save article flow (URL, notes, tags)
- Link library items to countries/scrapbook
- Wishlist page with add destination
- Planned countries on passport map (visual distinction)
- Pre-trip planning view

## Acceptance Criteria

- User can save articles/links to the Library with tags
- Library items can be linked to specific countries
- User can search and filter library items
- User can add countries to their Wishlist
- Wishlisted countries appear distinctly on the Passport map
- User can create a pre-trip checklist for wishlisted countries
