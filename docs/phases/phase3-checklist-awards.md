# Phase 3: Checklist & Awards System

## Backend (Go)

- Checklist items API (predefined + custom items)
- Activity categories (Cultural, Nature, Food, etc.)
- Completion tracking with timestamps
- Awards/badges model and unlock logic
- Progress calculation endpoints

## Frontend (Angular)

- Checklist page with category filters
- Expandable checklist item component
- Mark complete flow with confirmation
- Awards page with progress bars
- Badge unlock animations/notifications
- Integration: checklist completion -> passport stamps

## Acceptance Criteria

- User can view checklist items organized by category
- User can mark checklist items as complete
- Completing items adds stamps to the corresponding country in Passport
- Awards page shows progress bars for each badge category
- Completing all items in a category unlocks a badge
- Badge unlock triggers a celebratory notification
